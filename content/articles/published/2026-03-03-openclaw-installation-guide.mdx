# OpenClaw 完全安装指南

> 从零开始安装配置 OpenClaw，包含 macOS、Linux、Windows 全平台教程

---

## 一、系统要求

### 最低配置
- **Node.js**: 18.0 或更高版本
- **内存**: 2GB RAM
- **磁盘**: 500MB 可用空间

### 推荐配置
- **Node.js**: 20.x LTS
- **内存**: 4GB+ RAM
- **磁盘**: 1GB+ 可用空间

---

## 二、macOS 安装

### 方式一：使用 Homebrew（推荐）

```bash
# 安装 Node.js
brew install node

# 安装 OpenClaw
npm install -g openclaw

# 验证安装
openclaw --version
```

### 方式二：使用安装脚本

```bash
curl -fsSL https://install.openclaw.ai | bash
```

### macOS 特有配置

**允许通知权限**：
系统设置 → 通知 → OpenClaw → 允许

**自启动配置**：
```bash
# 创建 LaunchAgent
mkdir -p ~/Library/LaunchAgents
cat > ~/Library/LaunchAgents/com.openclaw.gateway.plist << 'EOF'
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.openclaw.gateway</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/local/bin/openclaw</string>
        <string>gateway</string>
        <string>start</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
</dict>
</plist>
EOF

# 加载
launchctl load ~/Library/LaunchAgents/com.openclaw.gateway.plist
```

## 三、Linux 安装

### Ubuntu/Debian

```bash
# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 OpenClaw
sudo npm install -g openclaw

# 创建 systemd 服务
sudo tee /etc/systemd/system/openclaw.service > /dev/null << EOF
[Unit]
Description=OpenClaw Gateway
After=network.target

[Service]
Type=simple
User=$USER
ExecStart=/usr/bin/openclaw gateway start
Restart=on-failure

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
sudo systemctl enable openclaw
sudo systemctl start openclaw
```

### CentOS/RHEL

```bash
# 安装 Node.js
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -
sudo yum install -y nodejs

# 安装 OpenClaw
sudo npm install -g openclaw
```

## 四、Windows 安装

### 使用 PowerShell

```powershell
# 安装 Node.js (使用 fnm)
fnm install 20
fnm use 20

# 安装 OpenClaw
npm install -g openclaw

# 验证
openclaw --version
```

### 使用安装包

1. 下载 `.msi` 安装包：[下载链接](https://openclaw.ai/download)
2. 双击运行安装向导
3. 完成安装后打开 PowerShell 验证

### Windows 服务配置

```powershell
# 使用 nssm 创建服务
nssm install OpenClawGateway
# 在 GUI 中配置：
# Path: C:\Program Files\nodejs\openclaw.cmd
# Arguments: gateway start
```

## 五、初始化配置

### 第一步：创建配置文件

```bash
# 创建配置目录
mkdir -p ~/.openclaw

# 初始化配置
openclaw config init
```

### 第二步：配置 AI 模型

```bash
# 交互式配置
openclaw configure

# 或手动编辑配置文件
cat ~/.openclaw/openclaw.json
```

### 示例配置

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "kimi-coding/k2p5"
      }
    }
  },
  "gateway": {
    "port": 18789,
    "auth": {
      "mode": "token"
    }
  },
  "channels": {
    "telegram": {
      "enabled": false
    }
  }
}
```

## 六、启动 Gateway

```bash
# 前台启动（调试用）
openclaw gateway start

# 后台启动
openclaw gateway start --daemon

# 查看状态
openclaw gateway status

# 停止
openclaw gateway stop
```

## 八、常见问题

### Q1: 安装时权限错误

**解决**: 使用 `sudo` 或修复 npm 权限
```bash
sudo chown -R $(whoami) ~/.npm
```

### Q2: 端口被占用

**解决**: 更换端口或关闭占用程序
```bash
openclaw configure --port 18790
```

### Q3: 无法连接 AI 服务

**解决**: 检查网络连接和 API Key
```bash
openclaw doctor --verbose
```
