# OpenClaw 搜索功能完全使用指南

> 快速获取网络信息，让你的 AI 助手更智能

---

## 简介

OpenClaw 内置了两款强大的搜索工具，让你无需离开对话即可获取网络信息：

| 工具 | 功能 | 适用场景 |
|------|------|----------|
| `web_search` | 网络搜索（Brave Search API） | 查找资料、研究话题、获取最新信息 |
| `web_fetch` | 网页内容提取 | 阅读文章、获取页面详情、分析网页内容 |

---

## 一、Web Search 工具

`web_search` 使用 Brave Search API，提供高质量的搜索结果。

### 基础用法

```
搜索 "Python 异步编程最佳实践"
```

只需直接提出搜索需求，OpenClaw 会自动调用 `web_search` 工具。

### 高级参数

- **region**: 指定搜索区域（如 'CN', 'US'）
- **language**: 搜索结果语言
- **freshness**: 时间过滤（'pd'=过去一天, 'pw'=过去一周, 'pm'=过去一月, 'py'=过去一年）

### 实际案例

**场景**: 了解最新的 AI 发展趋势

```
搜索 "2024年 AI Agent 发展趋势"
```

OpenClaw 会返回最新的搜索结果，帮你快速了解行业动态。

---

## 二、Web Fetch 工具

`web_fetch` 用于提取指定网页的内容，支持 HTML 转 Markdown。

### 基础用法

```
帮我提取 https://example.com/article 的内容
```

### 使用场景

1. **阅读长文**: 提取文章主要内容，无需打开浏览器
2. **资料整理**: 收集多个网页的信息进行对比
3. **内容分析**: 分析网页结构、关键词等

---

## 三、最佳实践

### 1. 搜索 + 提取组合使用

```
搜索 "OpenClaw 使用教程"，然后提取第一个结果的详细内容
```

### 2. 多源验证

对重要信息，使用多个搜索词交叉验证。

### 3. 指定时间范围

获取最新信息时，添加时间过滤：

```
搜索 "React 19 新特性 freshness:pw"
```

---

## 四、常见问题

**Q: 搜索结果不准确怎么办？**  
A: 尝试使用更具体的关键词，或添加英文关键词。

**Q: 网页提取失败？**  
A: 部分网站有反爬机制，尝试其他来源或手动复制内容。

## 总结

OpenClaw 的搜索功能让信息获取变得简单高效。善用 `web_search` 和 `web_fetch`，让你的 AI 助手更加强大！
