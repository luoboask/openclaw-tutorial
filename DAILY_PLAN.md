## 📋 每日任务模板

### claw-collect (信息采集员) - 每日任务

**上午 09:30-10:30**

- [ ] **检查 OpenClaw 更新**
  - GitHub releases 页面
  - 查看最新 commit
  - 记录版本变化

- [ ] **监控社区动态**
  - Discord 讨论热点
  - GitHub issues (新提交的问题)
  - 用户常见问题整理

- [ ] **竞品跟踪**
  - 其他 AI 网关项目动态
  - 学习可借鉴的内容

**输出**: `memory/YYYY-MM-DD-collect.md`

---

### claw-article (内容创作员) - 每日任务

**上午 10:30-12:00 & 下午 13:30-14:00**

- [ ] **内容创作**
  - 根据昨日收集的信息写文章
  - 或更新现有文档
  - 每天至少 500 字或 1 个页面

- [ ] **质量检查**
  - 自测所有步骤
  - 检查错别字
  - 确保格式正确

**输出**: `workspace/articles/YYYY-MM-DD-{标题}.md`

---

### claw-code (开发运维员) - 每日任务

**下午 14:00-16:00**

- [ ] **内容发布**
  - 将文章发布到网站
  - 更新导航/链接
  - 本地构建测试

- [ ] **技术维护**
  - 修复昨日发现的 Bug
  - 优化性能/体验
  - 更新依赖（如有安全更新）

- [ ] **Git 提交**
  - 提交今日更改
  - 写清晰的 commit message
  - push 到仓库

**输出**: 代码提交 + 部署日志

---

### claw-admin (管理者) - 每日任务

**全天穿插 + 16:00-17:00**

- [ ] **晨会 (09:00)**
  - 5分钟同步昨日完成
  - 确定今日重点

- [ ] **过程监控**
  - 随时回答团队成员问题
  - 协调资源
  - 解决阻塞

- [ ] **审核发布 (16:00)**
  - 审核文章内容
  - 测试网站功能
  - 确认发布

- [ ] **日报 (17:00)**
  - 总结今日完成
  - 更新任务看板
  - 规划明日任务

**输出**: `memory/YYYY-MM-DD-daily-log.md`
