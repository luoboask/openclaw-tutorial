# 2026-02-17 样式修复记录

## 问题
`/docs/quickstart` 等文档子页面出现白底白字问题（文字颜色太浅，在白色背景上难以阅读）。

## 原因
文档子页面使用了 `prose prose-slate` 类，但没有外层容器设置背景色，导致内容区域与灰色背景融合，文字显示不清晰。

## 修复方案
为所有文档子页面的内容容器添加明确的白色背景和样式：

```diff
- className="prose prose-slate max-w-none"
+ className="prose prose-slate max-w-none bg-white p-8 rounded-xl shadow-sm"
```
